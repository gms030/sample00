| Command                   | Meaning                                 |
| ------------------------- | --------------------------------------- |
| `git checkout feature`    | Switch to the existing `feature` branch |
| `git checkout -b feature` | Create a new branch + switch to it      |
| `git branch`              | List all branches                       |
